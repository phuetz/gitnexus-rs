Problemes critiques DOCX

CRITIQUE Desktop : Namespaces XML insuffisants

Le Desktop ne declare que xmlns:r et xmlns:w. Word 2016 et versions anterieures peuvent rejeter le fichier. Le CLI declare 10 namespaces. Il faut copier les declarations du CLI vers le Desktop.

CRITIQUE Desktop : Pas de support listes imbriquees

Le CLI detecte l'indentation (2+ espaces ou tab) pour les sous-listes et genere ilvl="1". Le Desktop ignore l'indentation. De plus, numbering.xml du Desktop ne definit qu'un seul niveau.

HAUTE Les liens ne sont pas des hyperlinks Word

Les deux generateurs rendent les liens [text](url) comme du texte bleu souligne, mais sans element <w:hyperlink> ni relation dans document.xml.rels. L'utilisateur ne peut pas cliquer dessus.
